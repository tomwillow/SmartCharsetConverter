School Daysのシーンファイルに含まれている動画・音声・曲をwmvにエクスポートするプログラム

○動作環境
　Microsoft .NET Framework Version 1.1が必要です。
　コンソールアプリケーションです(突貫でつくったので‥)

○概要
　・School Daysのシーンファイルに含まれている動画・音声・曲を指定されたシーンファイルの
　　ディレクトリにwmvファイルでエクスポートします。
　・出力されるファイル名はwmvヘッダにあるファイルＩＤで作成されます。
　・動画・音声・曲はそれぞれ別々のwmvファイルになっている様です。
　・新しいアップデータが出ても再度エクスポートし直せば大丈夫だと思います。
　・動作確認は「修正プログラム v1.05A」で行っています。

○使用方法
　sd_exp.exe [*.gpk]

　例：
　　sd_exp SCENE.GPK.010
